## 📱 **HOW TO RUN THE APP**

1. **Install Dependencies** (if needed):

   ```bash
   npm install
   ```

2. **Start the App**:
   ```bash
   npm start
   # or
   npx expo start
   ```

3. **Choose Platform**:
   - Press `w` for web
   - Press `a` for Android
   - Press `i` for iOS
   - Scan QR code with Expo Go app
